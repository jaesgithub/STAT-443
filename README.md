# STAT 443: Time Series and Forecasting (Spring 2026)
This repository contains materials for UBC STAT 443 202 Winter 2026, focusing on time series analysis and forecasting techniques using R.

## Course Description
STAT 443 covers trend and seasonality, autocorrelation, stationarity, stochastic models, exponential smoothing, Holt-Winters methods, Box-Jenkins ARIMA/ARMAX approaches, and frequency domain analysis.
Learning objectives emphasize building statistical tools for data science, data handling, coding forecast measures, and understanding models like "All models are wrong, but some are useful."
Trend and seasonality, autocorrelation, stationarity, stochastic models, exponential smoothing, Holt-Winters methods, Box-Jenkins approach, frequency domain analysis.
